# Agent 工具调用测试 Demo - 项目总结

## 📋 项目概述

本项目是一个完整的 Agent 工具调用演示系统，基于 `pi-ai-go/agent` 包构建，展示了如何创建能够调用外部工具的智能助手。

### 核心特性

✅ **多工具支持** - 内置 4 种实用工具  
✅ **交互式对话** - 支持实时多轮对话  
✅ **自动化测试** - 包含完整的测试套件  
✅ **事件驱动** - 实时监控 Agent 状态  
✅ **易于扩展** - 简单的工具添加机制  
✅ **详细文档** - 完整的使用指南和示例

## 🛠️ 内置工具

### 1. 计算器 (calculator)

**功能**: 执行数学计算

**支持**:
- 基本运算: `+`, `-`, `*`, `/`
- 数学函数: `sqrt`, `sin`, `cos`, `tan`, `log`

**示例**:
```
计算 123 + 456 * 789
计算 sqrt(144)
计算 sin(30)
```

### 2. 天气查询 (weather)

**功能**: 查询城市天气信息

**返回**:
- 温度、天气状况
- 湿度、风速
- 更新时间

**示例**:
```
查询北京的天气
上海今天天气怎么样
```

### 3. 数据库查询 (database_query)

**功能**: 模拟数据库查询

**支持查询**:
- `user_info` - 用户信息
- `order_list` - 订单列表
- `product_info` - 商品信息

**示例**:
```
查询用户信息
查看订单列表
查询商品详情
```

### 4. 搜索工具 (search)

**功能**: 网络信息搜索

**返回**:
- 相关文章列表
- 文章标题、链接、摘要

**示例**:
```
搜索机器学习最新进展
查找 Go 语言教程
```
