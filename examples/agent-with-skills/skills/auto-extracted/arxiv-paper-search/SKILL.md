# arxiv-paper-search

Search arxiv for latest papers on a specified topic, iteratively refine queries to overcome broad-match dilution, and present results with structured summaries plus curated highlights.

## Trigger conditions

> Use this skill when a user wants to find recent academic papers on arxiv about a specific topic, especially when generic keyword searches return off-topic results dominated by mainstream fields. Also use when the user asks for "latest papers", "what's new", or "literature review" tied to arxiv.

## Steps

1. **Parse the search intent** — extract the core topic, any sub-domain restrictions, and the desired time range (default to recent months if unspecified).
2. **Start with API-based search** — use the arxiv API with a focused query that includes domain-specific category filters (e.g., `cs.CY` for education-and-computing papers) rather than broad keyword matches across all categories.
3. **Assess result relevance** — if the top results are dominated by irrelevant mainstream fields, diagnose why (e.g., query too broad, ranking bias toward high-citation areas) and construct a refined query with tighter category or keyword constraints.
4. **Iterate until topic coverage is achieved** — repeat the query-refine-search loop until the returned papers predominantly match the target topic, recording the query variations tried.
5. **Extract structured metadata** — for each relevant paper, capture: title, authors, publication date, arxiv category, PDF link, and a complete (or near-complete) abstract.
6. **Classify and curate** — group papers thematically, identify standout papers by novelty or practical value, and prepare a clear summary table plus a curated-highlights section with brief rationale.
7. **Handle deep-dive requests** — when the user selects specific papers for closer reading, retrieve and present the full abstract (or other available metadata such as full author list and submission history) directly from the arxiv API.

## Output

- A curated list of papers with structured summaries (title, authors, date, categories, PDF link, abstract snippet).
- Thematic grouping and a highlights section explaining why certain papers stand out.
- Full-abstract retrieval for user-selected papers on demand.
- Query refinement log (implicit or explicit) showing how the final result set was reached.
